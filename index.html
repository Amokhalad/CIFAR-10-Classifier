CIFAR-10 Image Classifier

This project demonstrates a CIFAR-10 image classification model. You can view images from the dataset, see the model's predictions, and compare them with the actual labels.

    Training Method

    The model was trained using a ResNet architecture on the CIFAR-10 dataset. The training process involved data augmentation techniques such as random cropping and horizontal flipping to improve generalization. The model was optimized using the Adam optimizer, and cross-entropy loss was used as the loss function. The training was conducted over 100 epochs with a batch size of 64.
